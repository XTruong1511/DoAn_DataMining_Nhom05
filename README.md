# IT Job Market Analytics Vietnam

## 📊 Phân tích và Dự báo Thị trường Việc làm IT tại Việt Nam

### 🎯 Mục tiêu

- Phân tích thị trường việc làm IT
- Dự đoán mức lương dựa trên kinh nghiệm và kỹ năng
- Dashboard tương tác và API backend

# Cách chạy

## Mở cmd

- cd /d ...\it_job_market_analysis -> đường dẫn source code
- chạy python -m venv venv
- chạy venv\Scripts\activate
- chạy pip install -r requirements.txt
- chạy python run_pipeline.py
- chạy run_all.bat
